2. Work with Pointers

學習指標的基本操作與指標運算

1

宣告指標並指向變數

基本指標宣告 
int a = 10;
int *p;     // 宣告指向 int 的指標
p = &a;     // p 儲存 a 的位址
  • p → 存放的是位址（address）
  • *p → 存放該位址的值（value at address）
2

讀取指標與變數資訊

指標資訊輸出 
printf("p = %p\n", p);     // p 目前指向的位址
printf("*p = %d\n", *p);   // p 所指向的值
printf("&p = %p\n", &p);   // p 自己的位址
💡 重要概念

p 指向哪個變數，*p 就代表該變數的值。

3

讓指標指向不同變數

指標重新指向 
int b = 20;
p = &b;    // 改為指向 b
  • 改變 p 指向的位置不會影響 p 自己的位址（&p 不變）。
  • *p 會隨指向的變數不同而改變值。
4

指標運算（Pointer Arithmetic）

指標位址運算 
printf("p      = %p\n", p);     // 當前位址
printf("p + 1  = %p\n", p + 1); // 位址向後偏移 1 個 int
  • 如果 p 是 int*，那麼 p + 1 會跳過 4 bytes（因為 sizeof(int) = 4）。
  • 位址的變化與所指向型態的大小有關。
⚠️ 重要提醒

指標運算的步幅由指標所指向的資料型態決定。int* 每次 +1 會前進 4 bytes，char* 每次 +1 只前進 1 byte。
