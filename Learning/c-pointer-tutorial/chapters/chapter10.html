10. Pointers and 2-D arrays

二維陣列的指標表示法與存取方式

🔢

2D 指標陣列的使用

理解二維陣列在記憶體中的佈局以及如何使用指標來表示和操作它是 C 語言中的一個重要進階主題。

2-D陣列與指標陣列的宣告

宣告方式
int B[2][3];  // 二維陣列，2 列 × 3 欄
int (*p)[3] = B;  // p 是一個指向「包含 3 個 int 元素之陣列」的指標

指標與位址關係

對於二維陣列 B，不同表達式代表的意義如下：

表達式 說明 等價於
B 指向第一列的指標 &B[0]
*B 得到第一列陣列，並退化為指向該列首元素的指標 B[0] 或 &B[0][0]
B+1 指向第二列的指標 &B[1]
*(B+1) 得到第二列陣列，並退化為指向該列首元素的指標 B[1] 或 &B[1][0]
*(B+1)+2 指向第二列第三個元素的指標 &B[1][2]
*(*B+1) 對第一列的第二個元素解參考取值 B[0][1]
💡 補充說明：完整範例

為了讓這些抽象概念更具體，這是一個完整的 C 程式，您可以複製並執行，親自觀察各個表達式輸出的位址與值。

完整可執行範例
#include <stdio.h>

int main() {
    int B[2][3] = { {2, 3, 6}, {4, 5, 8} };

    printf("B or &B[0]           = %p\n", B);
    printf("*B or B[0] or &B[0][0] = %p\n", *B);
    printf("B+1 or &B[1]         = %p\n", B + 1);
    printf("*(B+1) or B[1]     = %p\n", *(B + 1));
    printf("*(B+1)+2 or &B[1][2] = %p\n", *(B + 1) + 2);
    printf("*(*B+1) or B[0][1]   = %d\n", *(*B + 1));

    return 0;
}
🔗

二維陣列的存取等價式

在 C 語言中，存取二維陣列的元素有多種等價的寫法，它們最終都會被編譯器解析為相同的記憶體位址操作。理解這些等價式有助於深入掌握指標的運作原理。

存取等價公式
B[i][j]  ==  * (B[i] + j)  ==  * ( * (B + i) + j )
  • B + i: 這是指標運算，B 是一個指向 int[3] 陣列的指標。B+i 會得到指向第 i 列陣列的指標。
  • *(B + i): 對指向第 i 列的指標進行解參考，會得到第 i 列的陣列。在表達式中，這個陣列名稱會再次退化為指向該列第一個元素（B[i][0]）的指標。
  • *(B + i) + j: 這是對指向 B[i][0] 的指標進行運算，使其指向該列的第 j 個元素。
  • *(*(B + i) + j): 最後，對指向 B[i][j] 元素的指標進行解參考，從而取得該元素的值。
