🎯 為什麼要學習指標？

指標是C語言最重要也是最具挑戰性的概念。掌握指標，就像拿到了程式設計世界的「萬能鑰匙」， 讓你能夠直接操作記憶體、提升程式效能、設計靈活的資料結構。

🚀 高效記憶體管理
學會動態配置記憶體，處理大型資料結構，避免記憶體洩漏，讓你的程式更加穩定高效。
🔧 靈活的函式設計
使用函式指標實現回呼函式，讓程式碼更模組化，實現更優雅的程式架構。
⚡ 提升程式效能
直接操作記憶體位址，減少不必要的資料複製，讓你的程式運行如飛。 
#include <stdio.h>

int main() {
    int a = 10;
    int *p = &a;     // p 指向 a 的位址
    
    printf("a 的值: %d\n", a);
    printf("a 的位址: %p\n", &a);
    printf("指標 p 的值: %p\n", p);
    printf("透過指標取得的值: %d\n", *p);
    
    return 0;
}
